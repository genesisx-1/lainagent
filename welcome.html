What is Kami Claw
Your AI Workforce, Locally Run

Kami Claw turns CLI-based AI models into a team of employees. Each agent has its own terminal, memory, and personality. No API keys on the frontend — everything runs through CLI commands.

⌘

CLI-First Architecture

Every agent runs off terminal commands — claude, gemini, codex, qwen, ollama. The app spawns and monitors processes. No SDKs, no API keys baked into the UI. Your CLI tools are the backend.

♻

Survives Shutdown

Agents write memory to disk as Markdown files. On restart, they pick up where they left off — conversations, tasks, context, everything. Shut your laptop, open it tomorrow, agents remember.

⚡

Cloud ↔ Local Swap

When cloud tokens run low, agents auto-swap to local LLMs via Ollama. Same agent, same memory, different brain. Claude swaps to DeepSeek. Gemini swaps to Gemma. Zero downtime.

⚒

Skills & Workflows

Reusable automations defined as simple Markdown files. Chain skills into multi-step pipelines — scrape leads, research them, write outreach, send emails. Build once, run forever.

Feature Breakdown
Everything Agents Need to Work
❖

Agent Creation via CLI

Create agents with a single command. Assign roles, departments, CLI backends, fallback models, and permissions. Each agent gets its own config file, memory directory, and terminal.

▶

Live Terminals Per Agent

Every agent has an embedded terminal showing real-time CLI output. Watch them think, see what commands run, and monitor their reasoning live. Tab between agents like browser tabs.

⇄

Inter-Agent Communication

Agents talk to each other through a shared message bus. Threaded conversations, priority levels, file attachments. Watch their conversations live or search history.

⚙

Task Pipeline System

Tasks flow through stages: inbox, queued, assigned, in progress, review, done. Auto-dispatch by keywords or agent workload. Dependencies, priorities, recurring tasks.

☍

Skills & Workflows

Define reusable skills as Markdown files with YAML frontmatter. Chain them into multi-step pipelines: scrape → research → score → outreach. Ships with 15+ built-in skills.

🌐

Browser Agent

Built on Playwright. Agents navigate websites, fill forms, scrape data, take screenshots, monitor competitors, and deploy websites. All visible in the terminal tab.

⬢

CRM & Lead Pipeline

Full CRM with contacts, leads, deals, companies. Visual pipeline: prospect → contacted → meeting → proposal → won. Agents work leads automatically.

⬡

Multi-Business

Run multiple businesses from one install. Each gets its own agent team, CRM, workflows, file storage, and knowledge base. Switch with a dropdown.

⏰

Heartbeat System

Every 30 minutes, agents review their context, check for events, write learnings to memory, and only alert you when something needs attention. Proactive, not reactive.

⬆

File & Document Manager

Upload files from your Mac or Telegram. Agents process PDFs, CSVs, images. Shared knowledge base per business. Auto-processing rules on upload.

☎

Telegram & Integrations

Text your head agent from Telegram. Connect Notion, Gmail, Google Calendar, Slack, Discord, GitHub, Stripe. Simple YAML config per platform.

☁

Deploy Anywhere

Run locally on your Mac, or deploy to a $5/mo VPS via Docker. Headless mode with REST API. Agents run 24/7, synced back to your desktop.
